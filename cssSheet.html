Text and Font Styling

Color

Sets the text color.
color:#007BFF;

font-family

Specifies the font type.
font-family: 'Times New Roman', Times, serif;

font-size

Sets the size of the font.
font-size:40px;

text-decoration

Adds underline, line-through, etc.
text-decoration: underline;



Box Model Properties

width

Specifies the width of an element.
width: 300px;

height

Specifies the height of an elelment.
height: 200px

margin

Sets the space outside the element.
margin: 60px;

padding

Sets the space inside the element.
padding: 30px;

border

Defines the border around an element.
border: 2px solid Coral;



Background Styling

background-color

Sets the background color.
background-color: rgb(20,20,20);

background-image

Sets a background image.
background-image: url('image.jpg');

background-repeat

Specifies whether and how the background image repeats
background-repeat: no-repeat;

background-position

Defines the starting position of the background image.
background-position:center bottom;



Display and Visibility

display

Controls how an element is displayed.
display: none;

visibility

Controls element visibility.
visibility: hidden;

The display and visibility properties in CSS are used to control the visibility and rendering of HTML elements, but they behave differently.

The Display Property

The display property determines how an element is rendered and whether it occupies space in the layout.
Key Points:
  • Setting display: none; removes the element entirely from the document flow.
  • It will not occupy any space in the layout.
  • Child elements of the element are also not rendered.
  • Default Display Values: Block-level elements (e.g., div, p) have display: block. Inline elements (e.g., span, a) have display: inline.

The Visibility Property

The visibility property hides the element visually but does not remove it from the document flow.
Key Points:
  • Setting visibility: hidden; makes the element invisible, but it still occupies space in the layout.
  • Child elements of the hidden element are also invisible but still occupy space.
  • The element remains part of the layout, so it affects positioning and space around it.
